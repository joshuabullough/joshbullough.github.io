Want To learn How to Code?

Welcome to my website! Here you will learn how to code webpages using HTML, CSS, and JavaScript at a very basic level. Each of the pages offer coding challenges for people who want to begin learning how to code.

HTML is the standard markup language for creating websites, CSS is the language that describes the style of an HTML document, and JavaScript is the language that brings on the functionality of the site. We will combine HTML, CSS, and JavaScript to create basic uses for building web pages.

I like to think of it as building a house. In this analogy, HTML is like the framework with all the walls, roof and siding. CSS is similiar to the design of the house, its in charge of the color of the carpet, the type of drapes and where the furniture goes. JavaScript is like the electrical work and makes it so your house can have light when you flip on a switch. So if you like building things, then you are in the right place!
